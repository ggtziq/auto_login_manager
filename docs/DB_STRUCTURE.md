# 資料庫結構 (SQLite)

資料表名稱: `password`

| 欄位名稱          | 型別  | 說明              |
|-------------------|-------|------------------|
| name              | TEXT  | 帳號 (主鍵)       |
| nameinput_path    | TEXT  | 帳號輸入框 XPath  |
| passwd            | TEXT  | 密碼              |
| passwdinput_path  | TEXT  | 密碼輸入框 XPath  |
| action            | TEXT  | 登入按鈕 XPath    |
| webway            | TEXT  | 網址              |
| webname           | TEXT  | 網站名稱          |
